IBM Telecom Customer Churn Prediction Dashboard

This project predicts whether a telecom customer is likely to churn using Machine Learning
and shows the results in an interactive dashboard.

Models used:
- Random Forest (for churn risk)
- Logistic Regression (for churn probability)

Dashboard features:
- Customer input panel
- Churn risk score
- Churn likelihood
- Reasons for churn
- Suggested actions to reduce churn
